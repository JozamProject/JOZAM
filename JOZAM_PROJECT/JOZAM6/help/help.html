User Manual

Notice

The JOZAM task manager is a local lightweight software that does not require an internet connection. It can be loaded on any web browser and on both operating systems WINDOWS and UNIX. However, so as to launch it, a server Apache needs to be activated first. It was designed to be simple, but still incorporates variable useful features:

  • Basic task management ( create or delete a task, every task has a name, a deadline, a description, etc…)
  • Each task can be divided into subtasks
  • Calculating the number of workdays left, by calendar integration
  • Display with folding / folding out options

Launching instructions:

In order to start JOZAM on Windows, please follow the following instructions:
First, run WAMPSERVER (the local server in question).
Second, open your web navigator, and type this URL : http://127.0.0.1/JOZAM/index.php

In order to start JOZAM on Ubuntu, please follow the following instructions:
First, Install Lamp-Server with the following command: sudo apt-get install lamp-server^
Second, put the project file in /var/www/html/
Then, execute the commands:

  • sudo chown -R $USER:$USER /var/www/html/folder-name
  • sudo chmod -R 777 /var/www/html

Finally, open your browser and type 127.0.0.1/folder-name/ The JOZAM software is then running, and its interface is displayed.
You can start by logging in with your username and password.
If you are logging in for the first time, click on the applet “Register” and fill in the requested input fields.

Operating instructions:

Once logged in, you can access to the main window. You can start by setting the language. For that, you need to click on the tab next to the “Disconnect” button and select the desired language.
You can configure new languages by:

  • adding in the file ”Languages.xml” the following line:
    ‹Language name="Language_Name" chosen="false"› language_name.xml‹/Language›
  • opening the file “language_name.xml” and filling up all the fields with appropriate inputs.
The second step is creating new boards: Click the “Add Board” tab in the top menu bar of your screen. Then, all the created boards are displayed and you can modify them :
  • Renaming a board : Click on the title of the board in question and type the new name in the corresponding field of the pop-up.
  • Adding a new project : In the corresponding board, click on the button “Add project”.
  • You can manually resize an existing project by clicking and dragging the bottom right corner of the project window.
  • To move a project, click on the top left corner logo and drag the project to the targeted area.
  • To move a project, click on the middle top triple stroke logo and drag the project to the targeted area.
  • To delete a project, click on the “cross” on the top right angle of the selected project.
  • To add a project inside another project, just click on the “plus” icon But you can also use the cut-and-paste shortcut: Ctrl+X on a selected project to cut it, select a board or a project and type Ctrl+V to paste there.
  • A copy-and-paste shortcut is also available to duplicate a project: double click on a project and type Ctrl+C to copy, then select a board or a project and type Ctrl+V to paste there.
  • To annex a task to a project, click on the asterix icon then you will have to fill in the following sections (title, description, deadline and comment) in a new pop up. Moreover, you can set the archive option for completed tasks. They will then be crossed out. This will enable you afterwards to use the “Archive” button to hide all these crossed out tasks and store them in the file “archive.txt”.
  • To toggle between boards, you can click on the “Toggle” applet and then select boards you wish to hide. Or, you can also use the ALT+T keyboard shortcut.
  • To save the current modifications, click on the button “synchronize back up files”.
  • To restore the latest backup files, click on the “Roll back” button.
